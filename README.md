# AI Expense Tracker Flutter App

AI-powered expense management application built with Flutter.

## Features

- Add and manage expenses
- Expense categories
- Spending analytics
- AI-based spending insights
- Receipt scanning
- Local expense storage

## Tech Stack

- Flutter
- Dart
- Isar Database
- Google ML Kit
